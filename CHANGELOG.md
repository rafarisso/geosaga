# Changelog

## v1.0.0 - MVP funcional de quiz

Versão estável criada antes da implementação do modo de plataforma e combate 2D.

### Incluído

- tela inicial responsiva para desktop e celular;
- seleção das cinco regiões brasileiras;
- cinco guardiões com assets processados e transparentes;
- quiz com cinco perguntas por região;
- feedback didático, pontuação e badges;
- progresso salvo no `localStorage`;
- layout móvel em retrato e paisagem;
- publicação funcional na Netlify.

### Referências

- Site: https://geosaga-brasil.netlify.app
- Repositório: https://github.com/rafarisso/geosaga

### Restauração

Para recuperar exatamente esta versão:

```bash
git switch --detach v1.0.0
```

Para criar uma nova branch baseada nela:

```bash
git switch -c restaurar-v1.0.0 v1.0.0
```
